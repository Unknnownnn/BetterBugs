# BetterBugs

BetterBugs is an early-stage bug capture platform that collects browser context (console, network, errors, state) and supports replay/triage workflows.

Status
- Early development. Core capture flow and API are under active build.
- See SETUP.md for detailed local setup.

Quick start
1) Start local services (MongoDB + MinIO) and the API.
2) Build and load the extension in Chrome.
3) Set project key (default dev-project/dev-key for local dev).

See setup guide: SETUP.md
